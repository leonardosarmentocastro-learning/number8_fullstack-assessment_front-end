### Getting started

Ensure to have [Node.js LTS](https://github.com/nvm-sh/nvm) and install project dependencies:

```shell
npm install
```

Proceed to start the development server:

```shell
npm run dev
```

Open the browser of your preference and navigate to:

```
http://localhost:3000/
```
